Код служит для экспериментальной проверки теоретического базиса работы "Система анализа метаинформации музейного фотоисточника".
Включает код парсинга базы данных Госкаталога.РФ (С Портала открытых данных Министерства Культуры РФ), а также код семантической разметки изображений с сопроводительными иллюстрациями и ссылками.

Файлы:
photoonto.json|photoonto.svg|photoonto.ttl - файлы общей схемы рекомендованного поискового графа, созданного для отражения сущностей внутри изображения (здесь - музейного фотоисточника). Узлы (классы) представляют точки интереса исследователя в музее и сущности для разметки изображения, а отношения между ними - важную для задач поиска связь.

get_and_processing.ipynb - код обработки данных Госкаталога и получения набора данных для выборки
test_clearing.ipynb - код препроцессинга изображений (улучшение качества)
bulla.ipynb - обработка коллекции изображений и проведение семантической сегментации, сохранение результатов
viewer.ipynb - просмотровщик, позволяющий выводить два изображения (с сегментацией и без) для сравнения 
df_double_clear_finish.pickle - датафрейм с метаданными изображений (фотографии имеют лицевую и оборотную сторону)
df_one_clear_finish.pickle - датафрейм с метаданными изображений (фотографии имеют только лицевую сторону)
data_photosourse.json - структура данных
