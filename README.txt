Код служит для экспериментальной проверки теоретического базиса работы "Система анализа метаинформации музейного фотоисточника".
Включает код парсинга базы данных Госкаталога.РФ (С Портала открытых данных Министерства Культуры РФ), а также код семантической разметки изображений с сопроводительными иллюстрациями и ссылками.

Файлы:
photoonto.json|photoonto.svg|photoonto.ttl - файлы общей схемы рекомендованной онтологии, созданной для отражения сущностей внутри изображения (что в нашем частном случае является музейным фотоисточником). Узлы (классы) представляют точки интереса исследователя в музее и классы для разметки изображения, а отношения между ними - важную для задач поиска связь.
